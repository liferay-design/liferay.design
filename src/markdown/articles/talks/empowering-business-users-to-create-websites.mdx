## Summary

Crear experiencias digitales innovadoras de forma ágil es cada vez más importante como factor diferenciador para las empresas en todas las industrias. Debido a eso, los usuarios de negocio a menudo confían en soluciones que les brinden más autonomía para crear soluciones y adaptarlas con el tiempo.

Esta sesión muestra cómo un ambicioso conjunto de funcionalidades introducidas en Liferay 7.2 proporciona a los usuarios de negocio más autonomía para crear landing pages, un sitio para un evento, un conjunto de páginas para un producto y mucho más. Además, Liferay 7.2 también permite a los desarrolladores ofrecer las herramientas exactas que requieren los usuarios de negocio para resolver sus necesidades específicas de manera efectiva.

Esta sesión es particularmente interesante para los clientes y desarrolladores que trabajan con sistemas creados con Liferay 6.x, ya que la sesión mostrará cómo las capacidades WCM de Liferay han evolucionado significativamente. Además, cualquiera que haya estado siguiendo el proyecto Modern Site Building podrá ver el gran avance en 7.2.
